## About

See the [docs](http://agquick.github.io/quick_script).

## Versioning

1. Increment version in bower.json
2. Commit to git
3. Tag with new version number

		$ git tag -a v2.0.1 -m "Version 2.0.1"

4. Push code and tags

		$ git push github master --tags
